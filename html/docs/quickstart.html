Introduction

These quickstart guides will provide you with simple instruction on how to install Icinga from source and have it monitoring your local machine inside of 20 minutes. Advanced installation options and configuration are discussed elsewhere. Besides, a new quick install guide for using the IDOUtils of Icinga is also available.

Guides

Quickstart installation guides are currently available for Linux (most common Fedora, Ubuntu, openSUSE):

  • Icinga Quickstart
If you are planning to use Icinga with IDOUtils for writing into a database:
  • Icinga with IDOUtils Quickstart

If you are installing Icinga on an operating system or Linux distribution that isn't listed above, read the Icinga Quickstart for an overview of what you'll need to do. Command names, paths, etc. vary widely across different OSes/distributions, so you'll likely need to tweak the installation docs a bit to work for your particular case.
